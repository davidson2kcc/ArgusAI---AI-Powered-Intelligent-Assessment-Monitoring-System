ArgusAI is a real-time, privacy-aware assessment monitoring and proctoring platform designed for academic institutions and enterprise certification programs. Powered by client-side computer vision telemetry, lightweight rule analysis, and dynamic risk scoring, ArgusAI enables invigilators and faculty to monitor large candidate cohorts effortlessly. The platform captures dual-stream feeds—combining live desktop screen shares with corner webcam picture-in-picture streams—and analyzes real-time signals including eye gaze direction, head pose orientation, browser tab switching, and ambient audio levels to ensure academic integrity.

For faculty and proctors, the dashboard provides a stream layout featuring Extra Large Icon Grid and Priority Table views, allowing invigilators to scroll smoothly through up to 100 student sessions while maintaining visual clarity. Candidates are automatically sorted in real time by risk priority so that high-risk anomalies appear at the top. Non-blocking sticky alert banners notify faculty of sudden infractions without interrupting their ongoing monitoring workflow, and one-click candidate inspection modals allow proctors to review detailed telemetry logs and issue official malpractice warnings directly. Built on Express and modern ES6 JavaScript, ArgusAI delivers a low-latency, scalable proctoring solution with zero cognitive overload for educators.
